# Amazon Scraper Django App

A Django-based web scraper that extracts product details from Amazon India using BeautifulSoup and Requests.

## Features
- Submit Amazon product URL
- Scrapes title, price, and rating
- Saves results to CSV
- Downloadable output

## Tech Stack
- Python 3.14
- Django 5.2
- BeautifulSoup
- Requests
- Pandas

## How to Run
```bash
python manage.py runserver
